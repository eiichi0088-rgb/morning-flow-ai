# CHANGELOG

MORNING FLOW AI の各バージョンの変更内容を記録します。

## Version 1.0 - 2026-05-29

- 音声入力による文字起こしMVPを作成
- 大きなマイクボタン、認識中表示、文字起こし欄、やり直し、次へ進むボタンを追加
- スマホファーストの未来的UIを実装

## Version 1.2 - 2026-05-29

- ローカル解析で今日の目的、目標、TODO、優先順位、推奨スケジュール、AIアドバイスを生成
- サンプル入力ボタンを追加
- ガラスUIと未来的カードUIを強化

## Version 1.3 - 2026-05-29

- OpenAI APIによる実入力のAI整理に対応
- AI処理を `src/services/aiPlanner.ts` に分離
- 仕事、健康、家族、学習の4カテゴリー分類を追加
- `.env` によるAPIキー管理を追加

## Version 1.4 - 2026-05-29

- 昨日の振り返り画面を追加
- 達成チェック、達成率表示、AI振り返り、繰り越し機能を追加
- LocalStorage保存処理を `src/services/reflectionStorage.ts` に分離

## Version 1.5 - 2026-05-29

- `PROJECT_RULES.md` を作成
- 開発ルール、デザインルール、バージョン管理ルール、リリースルールを整理
- release時のbuild、ZIP、CHANGELOG、VERSION確認を自動化

## Version 1.6 - 2026-05-29

- AI Morning Coachを追加
- 朝の気分選択、最重要ミッション、3つの集中項目、成功条件を追加
- Vercel公開準備、README、公開手順、GitHub/Vercel運用ルールを追加

## Version 1.7 - 2026-05-30

- 音声認識後の文字起こし編集エリアを追加
- 修正を保存、元に戻すボタンを追加
- 入力中の内容をLocalStorageへリアルタイム保存

## Version 1.8 - 2026-05-30

- 「次へ進む」ボタンを画面下部に固定表示
- スクロール位置に関係なく次の操作へ進めるように改善
- スマホで親指操作しやすいフローティングボタンへ変更

## Version 1.9 - 2026-05-30

- AI整理結果のタイムスケジュールからカレンダー予定を作成する機能を追加
- 「カレンダーへ追加」ボタンを推奨タイムスケジュール欄に追加
- 登録前に予定確認画面でタイトル、開始時刻、終了時刻、メモを確認できるように改善
- Googleカレンダーで開くリンクとAppleカレンダー用 `.ics` 保存に対応

## Version 2.0 - 2026-05-30

- Google OAuthログイン機能を追加
- AI整理後の予定一覧確認画面にチェック選択を追加
- 選択した予定のみGoogleカレンダーへ直接登録できるように改善
- 登録項目としてタイトル、開始時間、終了時間、説明文、優先度を送信
- Googleアクセストークンを永続保存せず、実行中の一時状態で管理
- Googleカレンダー連携のログアウト機能を追加
- `.env.example` に `VITE_GOOGLE_CLIENT_ID` を追加
- READMEをVersion 2.0向けに更新
- 画面表示を `v2.0` に更新
- `morning-flow-ai-v2.0.zip` として保存

## Version 2.1 - 2026-05-30

- GoogleカレンダーURLに `ctz=Asia/Tokyo` を追加し、日本時間の予定がズレないように修正
- GoogleカレンダーURLとICS出力で不要なUTC `Z` に依存しない日時形式へ変更
- Google Calendar API直接登録でも `Asia/Tokyo` の日時として送信するように修正
- 「次へ進む」ボタンでAI整理実行またはカレンダー登録エリアへ自然に移動できるように改善
- Googleカレンダー登録UIで一括登録をメイン操作として明確化
- 個別リンクを「編集して登録」の補助操作として整理
- 予定選択チェックボックスに `id`、`label htmlFor`、`aria-label` を追加
- 文字起こし編集中の内容がAI整理へ反映されることを明示
- 画面表示を `v2.1` に更新
- `morning-flow-ai-v2.1.zip` として保存
